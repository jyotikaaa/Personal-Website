Android Application
This android application works as a depression detector.
It uses a machine learning approach to detect depression with python scripts as backend.
The user is made to take a mcq test based on the answers of which the machine learning algorithm can label the depression stage.
The three labels include: no depression, mild depression and severe depression.
My contribution in the project has been designing the front end with Android Studio IDE using java.
We are still working on this project as a part of our Design Engineering subject.

Website
We developed this website in a team as a part of Smart Gujarat for New India Hackathon.
Our project definition was to build a machine learning model to predict fair house prices.
As usually people seek help from brokers to find properties that suit their needs the best, they always end up paying the brokerage costs as well.
Our model pricted the actual cost of the property based on 5 basic factors: Locality, BHK, Area per Sq. feet, Construction status and City.
The backend coding was done entirely in python while the frontend coding was done using HTML, CSS, Bootstrap.
Jquery was used to call the python scripts on a button click event.
My contribution remained mostly in the frontend development again.
